# FWC responsive website

Open `index.html` to view the website and `admin.html` to manage key homepage copy.

The included CMS works without a server: edits are stored in the browser with `localStorage`. Use **Export JSON** to back up content and **Import JSON** to restore or move it to another browser.

For deployment, upload the whole `fwc-website` folder to Netlify, Cloudflare Pages, GitHub Pages, or any static web host. A production database-backed CMS can later replace the local storage layer without changing the visual site.
